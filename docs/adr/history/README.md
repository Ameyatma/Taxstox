# ADR History

Formal Architecture Decision Records will be stored in this directory as numbered markdown files.

## Naming Convention

`NNNN-brief-slug.md`

Example: `0001-use-modular-monolith.md`

## Statuses

- **Proposed** — Under review
- **Accepted** — Approved and in effect
- **Deprecated** — No longer relevant
- **Superseded** — Replaced by newer ADR

See [docs/governance/01-Chief-Architect.md](../../governance/01-Chief-Architect.md) for the complete ADR process.
